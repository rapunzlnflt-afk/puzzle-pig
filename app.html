Welcome to Puzzle Pig

A fun way for kids to learn about saving money. Watch your goal picture come to life as you save toward it.

  • Parents: Set up users and a 5-digit PIN first
  • Kids: Tap your name to jump in

Goals and Puzzles

Upload a picture of something you want to save for. It starts hidden behind Puzzle Pig and reveals from the bottom up as you save.

  • Use a photo of the real item you want
  • Set a savings goal amount and optional target date
  • Each time you add savings, more of the picture appears

Save, Spend, and Give Jars

Money goes into jars. The Save jar feeds your puzzle goals. You can also set up Spend and Give jars to learn about budgeting.

  • Parents choose which jars to turn on (PIN required)
  • Move money between jars with transfers
  • The Save jar balance powers your puzzle progress

Earn with Chores

Parents can set up chores that kids complete to earn money. Finished chores get approved by a parent, who chooses which jar the money goes into.

  • Parents create chores with a dollar amount (PIN required)
  • Kids tap "I Did It!" when a chore is done
  • Parents approve and pick which jar gets the money

You're All Set

Start by adding a user, then create your first savings goal. You can always tap the help button if you need a reminder about how things work.

  • Look for the ? button to get help on any page
  • Use the bottom bar to navigate Home and create New Goals
